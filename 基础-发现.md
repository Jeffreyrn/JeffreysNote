# commonJS vs import vs require

CommonJS 模块输出的是一个值的拷贝，ES6 模块输出的是值的引用

require是执行时加载，因此可以做到动态加载；而import是静态加载，是在编译阶段完成的加载，因此无法做到动态加载

# CMD vs AMD

``` javascript
//AMD方式定义模块
define(['dep1','dep2'],function(dep1,dep2){
     //内部只能使用制定的模块
      return function(){};
});
//CMD
define(function(require,exports,module){
   //此处如果需要某XX模块，可以引入
   var xx=require('XX');
});
```
